Instructions

In this exercise, you will create a flexible custom checkbox control that you can use for many different purposes. Note: p5.js already provides a checkbox control so you should generally use that if you need a checkbox unless you want to add styling.

Implement the checkbox

The following steps take you through the checkbox implementation.

  1. Create a new file called checkbox.js and connect it to this HTML file by adding the code below immediately after the <script> tag that links sketch.js: 
    <script src="checkbox.js"></script>
  2. For this implementation, you will use bounding box collision detection to detect when the user is interacting with the checkbox. Open up the example from lecture and copy the CollisionRect class into checkbox.js.
  3. Still in checkbox.js, create a new class called Checkbox, which is a child of CollisionRect. The class should have two private attributes, #label and #checked. The constructor should have parameters for the checkbox location and size, as well as a parameter for the label text, which should be used to set the initial value of the #label attribute. #checked should be false to begin with. Use the remaining constructor arguments and the appropriate parent class methods to set the location and dimensions of the collision rect. Try to implement this step on your own first, but you can click here to see a completed example.
  4. There are two ways to think about state for a checkbox: it's value (ticked or not), and it's interaction status. The #checked attribute will track whether or not the checkbox is ticked. Add a #state attribute and public static instance variables to represent its interaction status: disabled, enabled, and hover. It's starting state should be enabled.
  5. Add a draw() method that draws the checkbox (don't forget the label!). The checkbox should look different depending on its state and whether or not it is ticked. For example, you might change the colour scheme for each of the three states. The example below shows fairly standard looking checkboxes. Feel free to make yours more exiting! If you would like to use the standard tick (✓), copy and paste it from this text.
  6. Add a method called isChecked() that returns the #checked attribute.
  7. Add method(s) that set the state of the checkbox when it is hovered, disabled, or enabled. Use the custom button example as a guide.
  8. Finally, add a method that will toggle the value of #checked.
