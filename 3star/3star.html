Instructions

Virtual joysticks are commonly used to control characters in mobile games. Despite being fairly common, the virtual joystick is a non-standard control that developers often have to implement themselves.

Your task is to create a virtual joystick similar to the one in the video below. It should be possible to use the joystick to control the movement of a character.

Here are some tips and high level guidance for creating the control:

  • A virtual joystick consists of two components: a base (the larger circle), which does not move, and a handle (the smaller circle), which the user can drag around the base.
  • The handle needs collision detection to determine if the mouse is held down over the handle. A collision circle is recommended. You can find an example in the pie menu code shown in class.
  • The joystick should have two states: dragging and not dragging. When the state is "not dragging", the handle should be in the centre of the joystick. When the handle is being dragged, it should follow the mouse but it should not go beyond the joystick base. This is the most challenging part of this exercise! Some hints for the maths are provided in the hint below the video.
  • The joystick will have two values that could be used to control a character: an x value and a y value. Use the position of the handle to determine the values.

Expected output:

You can use the p5.js dist() function to figure out whether the mouse is out of bounds of the joystick base. The handle is in bounds if the distance between the handle centre and the joystick centre is less than the radius of the base minus the radius of the circle.

There are two ways to respond to the handle being dragged out of bounds:

  • Stop dragging the handle, set the joystick state to "not dragging" and set the handle position to be the centre of the joystick. This is the easiest option but the user experience is slighly worse.
  • Keep the joystick in "dragging" state but constrain the handle to the edge of the joystick, as shown in the video. The maths for this is simpler than you might think! Use the following calculations when the drag is out of bounds of the joystick:
    1. Use the dist() function to calculate the distance between the mouse coordinates and the joystick center. This is the current distance.
    2. Calculate the maximum distance the centre of the handle can be from the centre of the joystick: radius of the joystick - radius of the handle.
    3. Divide the current distance (step 1) by the maximum distance (step 2). We'll call this the proportional distance.
    4. Calculate the distance between the mouse x coordinate and the joystick x coordinate. Divide the result by the proportional distance (step 3): (mouse x - joystick centre x) / proportional distance.
    5. Add the result of step 4 to the joystick x coordinate. This is the adjusted x coordinate.
    6. Repeat steps 4 and 5 for the y coordinate. This is the adjusted y coordinate.
    7. Set the handle position using the adjusted coordinates.
