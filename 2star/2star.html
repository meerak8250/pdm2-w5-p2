Instructions

The goal of this exercise is to practice creating controls with animated transitions between states. You will create a toggle switch as shown in the video below. p5.js does not provide a built in toggle.

  1. Create a new file called toggle.js and connect it to the HTML as usual.
  2. Copy the CollisionRect class from your solution to the previous exercise into toggle.js and create a new Toggle class that inherits from CollisionRect.
  3. A toggle switch has two states from a user's point of view, on or off. To create the animated transition, your toggle will need four states: off, turning on, on, turning off. Add these to the class and set the inital state to off.
  4. Write a method to draw the control. Keep it simple for now: draw a rectangle for the background and a smaller shape for the switch. The switch should be on one side of the rectangle when the toggle is off and on the other side when the toggle is on. Add private instance variables to store the position of the switch—you will need these to create the animation later on.
  5. Add a method to flip the state of the switch when it is clicked. Don't worry about the animated states just yet. If the toggle is off when it is clicked, set the state to on, and vice versa. If you haven't done so already, create an instance of Toggle in the main sketch and write some code to test what you've implemented so far.
  6. Now you're ready to add the animated transitions. Add a private instance variable to store the switch speed. Speed should be zero when the toggle state is on or off. When the toggle enters an animated state (turning on or turning off), calculate the speed that the toggle should move:
    • speed = distance / time
    • distance = the toggle's final position - the toggle's start position. For example, if turning on: distance = toggle on position - toggle off position.
    • time = the number of frames you want the transition to last. 30 is a good starting point but you can tweak it until it looks right.
  7. Update the method you wrote in step 5 so that the state changes on click as follows:
    • If the toggle is off when clicked, set the state to turning on. Calculate the toggle speed.
    • If the toggle is on when clicked, set the state to turning off. Calculate the toggle speed.
    • If the toggle is turning on or off when clicked, do nothing.
  8. Write another method that moves the toggle swtich depending on state and the value of the speed attribute. If the toggle is in an animated state, the method should also check if the switch has reached its destination and update the state accordingly.
  9. Finally, add a method that allows users of the class to check if it is on or off. You will need to decide how to handle the animated states.

Add some more code to sketch.js to test the toggle e.g. move / stop a shape based on the state of the toggle. You may like to add some additional styling.
